# Archived Design Documents

Historical design specs kept for context. **None of the files in this directory are authoritative.** For the current game design, see `9LN_GAME_BIBLE.md` in the repo root.

Future historical docs should land here with a one-line entry in this README explaining what they are and roughly when they were superseded.

## Contents

- **`9LN_GAME_BIBLE_V3.md`** — Game bible through March 19 2026. Described Combat V3 (spatial, range-based targeting, continuous combat with a 1-minute cooldown). Superseded by V4 in April 2026, which introduced the round-based combat system, time-based DOT, and a full balance pass.
- **`ZONE_BATTLE_ROUNDS_V1.md`** — April 2026 design RFC for the round-based zone battle system. Replaced the continuous-combat model with discrete rounds that end when one guild wipes the others (or after a 5-minute hard cap). The design in this RFC was integrated into the V4 bible; this file is the record of the decision and its rationale.
