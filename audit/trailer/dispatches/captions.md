# Caption Library — Dispatches Wave 0

Post-copy for dropping each dispatch on Twitter / TikTok / IG Reels / YouTube Shorts. Five rotation options per dispatch so the same clip can be reposted across platforms (or re-posted on the same platform weeks later) with a fresh framing.

Read `/audit/trailer/launch/README.md` before posting. The hard editorial rules there — no emoji, no exclamation marks, no pump language, hashtag restraint, `@9LVNetwork` primary brand handle — apply verbatim to every line below.

## Voice mix in this library

Two voices in play:

- **`[nethara]`** — the dispatch narrator voice, as specified in `voice_canon.md`. Flat, institutional, world-as-subject. Use when the caption reads as an extension of the clip's VO.
- **`[account]`** — `@9LVNetwork` speaking directly, Nerm-adjacent voice (lowercase, no `!`, informational-but-restrained). Use when the caption frames the dispatch for viewers who need context.
- **`[nerm]`** — optional, only for a cross-promo tail-tweet (Nerm reacting to a Nethara dispatch). Not the primary voice here; cameo only.

Do NOT write captions that repeat the clip's VO line verbatim — the caption adds, it doesn't echo.

## Universal rules (every caption)

- Caption appears BELOW the video on most platforms. Read it as the world's footnote to the clip.
- Handle `@9LVNetwork` is optional per post. Include on fresh drops, drop on re-posts to vary the feed.
- No caption below ends in `!` or contains an emoji. Run the QC grep at the bottom of this file before any post.
- Hashtags: `#ninelivesnetwork` primary, plus one of `#nethara` or `#9LV` maximum. Never three hashtags on Twitter. TikTok can add `#fyp`.
- Do not reference other dispatches in captions until at least 3 have shipped — single-post context only in Weeks 1–2.

## Cross-dispatch rotation strategy

When dropping dispatches across a 2–3 week window, avoid:

- **Two calm-register captions back-to-back.** D-01 and D-04 both skew institutional; don't pair them on consecutive drops. Interleave with D-02 (glitch) or D-05 (cold imperative) to vary tone.
- **Nerm-cameo captions on successive drops.** Option 5 is Nerm across all 5 dispatches. Use Nerm captions sparingly — once per week at most — otherwise the dispatches read as "Nerm in disguise" and the Nethara voice loses distinctness.
- **Same caption across platforms.** Pick Option 1 for Twitter, Option 3 for TikTok, Option 4 for IG Reels — treat each platform as its own feed with its own rotation.

## Platform-specific adaptations

- **Twitter:** 280 char limit is not a concern for these (longest caption here is ~90 chars). Hashtags after caption body, separated by a blank line.
- **TikTok:** append hashtags directly to caption. `#fyp` acceptable as platform concession. Example: `the wall has been counted. it is still there. #ninelivesnetwork #nethara #fyp`
- **IG Reels:** caption 125–300 char ideal. Any of the options fit. Same hashtag rules as Twitter.
- **YouTube Shorts:** use the title slot instead of a caption. Max ~100 char. All options fit.
- **Twitter + pinned thread window:** when the master trailer launches and the pinned thread lives, dispatch drops should NOT include `@9LVNetwork` — that's the pinned thread's job. Caption speaks for itself during that week.

## QC before posting

- Read it aloud. Does it sound like Nethara or like a brand?
- Does it end with an exclamation mark? Fix it.
- Does it reference the viewer ("you," "we")? Cut or rewrite — Nethara voice doesn't address.
- Does it promise earning / gains / moon? Delete.
- Does it use an emoji? Delete.
- Is this specific phrasing already used in another active dispatch's caption? Rephrase.

```bash
grep -En '!' audit/trailer/dispatches/captions.md                       # zero hits in content
grep -EinH 'moon.{0,3}shot|wagmi|gm gm|lfg|ape |degen|pump|early ape' audit/trailer/dispatches/captions.md  # zero hits
```

(Note: "moon" alone is canon — it's a celestial body in-world, cf. D-05. "moonshot" is pump language. The regex above targets the latter.)

If either grep hits, rewrite.
